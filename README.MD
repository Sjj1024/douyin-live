# 抖音直播桌面版  
LiveBox是一个抖音直播桌面版本，输入抖音直播地址，就可以安安静静看直播和弹幕消息，拒绝消费，冷静看播。
特点：  
1.超级轻量，只有5M，不占内存，不占CPU，不占GPU，不占硬盘。
2.未来会增加多窗口支持，和登录支持，还可以自动化抢购和抢福袋等功能。  
仓库地址：https://github.com/Sjj1024/LiveBox  
桌面安装包下载地址：  
https://github.com/Sjj1024/LiveBox/releases
